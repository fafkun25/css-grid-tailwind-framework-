# 🏆 CSS Grid Mastery: Responsive Layout Gallery
A high-performance, responsive web design showcase built with **Tailwind CSS** and **CSS Grid**. 

This project serves as a "Developer Cheat Sheet," demonstrating six essential modern web layouts—from the classic Holy Grail to the dynamic Bento Box—all optimized for every device from mobile to desktop.

## 🚀 Live Demo
https://fafkun25.github.io/css-grid-tailwind-framework-/

## ✨ Key Features
* **Intelligent Auto-Grid:** Uses `grid-cols-[repeat(auto-fit,minmax(320px,1fr))]` to magically rearrange cards without complex media queries.
* **6 Modern Templates:**
    1. **Auto-Grid:** Fluid scaling for image galleries.
    2. **Bento Box:** Sleek, modular dashboard layout.
    3. **Holy Grail:** The standard 3-column web structure.
    4. **Sidebar-Fluid:** Fixed navigation with expanding content.
    5. **Skeleton:** Structural blueprints for wireframing.
    6. **Asymmetrical:** Artistic, rule-breaking grid spans.
* **Fully Responsive:** Tested across mobile, tablet, and desktop viewports using a Mobile Simulator.
* **Interactive UI:** Hover effects and scaling transitions for a premium "app-like" feel.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for SEO and accessibility.
* **Tailwind CSS:** For rapid, utility-first styling and layout management.
* **CSS Grid:** The core 2D layout engine.

## 🧠 What I Learned
During the development of this project, I focused on:
1.  **Muscle Memory:** Moving beyond rote memorization to fluidly typing layout logic.
2.  **Viewport Management:** Solving container alignment issues (the `mx-auto` fix).
3.  **Efficiency:** Balancing standard CSS fundamentals with the speed of frameworks.
