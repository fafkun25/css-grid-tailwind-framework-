What it is?

A modern, tile-based layout where items have different sizes (some span 2 columns, some span 2 rows). It looks like a Japanese lunch box.

Best used for?

Personal portfolios, feature highlights on a landing page, or Apple-style "summary" sections.

Best used for?

in any project you like.
